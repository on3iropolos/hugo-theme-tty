# Developer

Welcome to my digital terminal. I'm a software engineer who believes in simplicity, automation, and the Unix philosophy.

## What I Do

I build software that works. Web applications, APIs, automation scripts, and infrastructure. I prefer static over dynamic, simple over complex, and explicit over implicit.

## Tech Stack

- **Languages**: Go, Python, JavaScript, Rust
- **Web**: Hugo, Next.js, Express
- **Cloud**: AWS, Docker, Kubernetes
- **Tools**: Git, Vim, Linux

## Philosophy

1. **Ship often** - Small iterations beat big bangs
2. **Document less, code more** - Self-documenting code > elaborate docs
3. **Automate repetition** - If you do it twice, script it
4. **Read the source** - Documentation lies, code doesn't
